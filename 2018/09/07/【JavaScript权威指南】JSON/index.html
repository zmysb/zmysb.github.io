2006年，JSON（JavaScript Object Notation，JavaScript对象表示法）是JavaScript的一个严格的子集，利用了JavaScript中的一些模式来表示结构化数据。与XML相比JSON是在JavaScript中读写结构化数据更好的方式。因为可以把JSON直接传给eval()，而且不必创建DOM对象。

关于JSON，最重要的是理解它只是一种数据格式，不是一种编程语言。虽然具有相同的语法形式，但JSON并不从属于JavaScript。而且，并不是只有JavaScript。而且，并不是只有JavaScript才使用JSON，毕竟JSON只是一种数据格式。很多编程语言都有针对JSON的解析器和序列化器。
