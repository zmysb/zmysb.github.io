课程目标

​ 通过大量练习，来学习布局的各种方式

​ 重要先决知识，与布局密切相关的属性 ：

​ display ：block／inline／inline-block／flex …

​ 是CSS中最重要的用于控制布局的属性。每个元素都有一个默认的 display 值，这与元素的类型有关。对于大多数元素它们的默认值通常是 block 或 inline 。一个 block 元素通常被叫做块级元素。一个 inline 元素通常被叫做行内元素

​ box-sizing：人们慢慢的意识到传统的盒子模型不直接，所以他们新增了一个叫做 box-sizing 的CSS属性。当你设置一个元素为 box-sizing: border-box; 时，此元素的内边距和边框不再会增加它的宽度

定位

定位方式 position属性值 描述
静态定位（默认） position: static; 静态定位是每个元素获取的默认值——它只是意味着“将元素放入它在文档布局流中的正常位置
相对定位 position: relative; 与静态定位非常相似，占据在正常的文档流中， 你仍然可以修改它的最终位置，包括让它与页面上的其他元素重叠，通过对它自己在文档流中的本来位置设定偏移
绝对定位 position: absolute; 绝对定位的元素不再存在于正常文档布局流中； 绝对定位的元素相对于其定位的元素，通过在元素的其他祖先之一上设置定位来实现的；元素重叠通过z-index设置覆盖优先级
固定定位 position: fixed; 与绝对定位的工作方式完全相同，只有一个主要区别：绝对定位固定元素是相对于 元素或其最近的定位祖先，而固定定位固定元素则是相对于浏览器视口本身
粘性定位 position: sticky 基本上是相对位置和固定位置之间的混合，其允许定位的元件像它被相对定位一样动作，直到其滚动到某一阈值点（例如，从视口顶部10像素），之后它变得固定

弹性盒子

一些布局难以用floats和 positioning实现，弹性盒子使得很多布局任务变得更加容易

flex模型说明：

当元素表现为 flex 框时，它们沿着两个轴来布局：

表现为柔性的盒子的元素（flex项）设定 flex/flex-xxx等属性

其父元素（flex容器）设定 display: flex;

flex项属性 值 描述
flex-direction rou（默认）／column／ row-reverse／ row-reverse 指定主轴的方向（flex项依次放置）
flex-wrap wrap／wrap-reverse 换行
flex（缩写） 0 1 auto （默认） 以指定最多三个不同值的缩写属性：第一个flex-grow无单位比例用于控制flex项占用空间比例（主轴方向）；第二个无单位比例 flex-shrink一般用于溢出容器的 flex 项；第三个flex-basis控制flex项占最小值（主轴方向 ）
order 0（默认） flex 项排序： order 值大的 flex 项比 order 值小的在显示顺序中更靠后（主轴方向）
flex容器属性 值 描述
align-items stretch（默认）／center 控制 flex 项在交叉轴上的位置： 默认 stretch，其会使所有 flex 项沿着交叉轴的方向拉伸以填充父容器。如果父容器在交叉轴方向上没有固定宽度（即高度），则所有 flex 项将变得与最长的 flex 项一样长（即高度保持一致）； center 值会使这些项保持其原有的高度，但是会在交叉轴居中
align-self flex-start（默认）／flex-end 控制 flex 项在主轴上的位置： 默认值是 flex-start，这会使所有 flex 项都位于主轴的开始处； 你也可以用 flex-end 来让 flex 项到结尾处
justify-content space-around／center／ space-between 控制 flex 项在主轴上的位置：center 让 flex 项在主轴居中； space-around使所有 flex 项沿着主轴均匀地分布，在任意一端都会留有一点空间； space-between均匀分布但两侧不留下空间

对齐（对子元素而言）

  1. 水平居中

    (1) 行内元素水平居中
    (2)块级元素水平居中（定宽）
    (3)块级元素水平居中（不定宽）

    可以直接给不定宽的块级元素设置text-align：center来实现，也可以给父元素加text-align:center 来实现居中效果

  2. 垂直居中

    条件：父元素是盒子容器；父元素高度确定

    (1)行内元素垂直居中
    (2)块级元素垂直居中（定高）

    计算子元素的margin-top或margin-bottom，计算方法为父(元素高度-子元素高度)/2

    (3)块级元素垂直居中（不定高）

    子元素高度不确定，无法计算得到

  3. 水平垂直居中

    (1)水平对齐+行高

    text-align + line-height 实现单行元素水平垂直居中

    (2)水平+垂直对齐

    text-align + vertical-align 在父元素设置text-align和vertical-align，并将父元素设置为table-cell元素，子元素设置为inline-block元素

    (3)相对+绝对定位

    使用absolute，利用绝对定位元素的盒模型特性，在偏移属性为确定值的基础上，设置margin:auto

    (4)flex模型

  1. 一列布局

  2. 两列布局

    ​ 两栏布局的七种方法。其中有老生常谈的float方法,BFC方法，也有CSS3的flex布局与grid布局。并非所有的布局都会在开发中使用，但是其中也会涉及一些知识点。详解可以查看这里

    ​ 常用的宽度自适应的方法通常是利用了block水平的元素宽度能随父容器调节的流动特性。另外一种思路是利用CSS中的calc()方法来动态设定宽度。还有一种思路是，利用CSS中的新型布局flex layout与grid layout。

    (1)双 inline-block方案
    (2)双float方案
    (3)float+margin-left方案
    (4)使用absolute+margin-left方法
    (5)使用float+BFC方法
    (6)flex方案
    (7)gird方案

  3. 三列布局

    两侧定宽中间自适应

    (1) 圣杯布局

    圣杯布局，为了中间div内容不被遮挡，将中间div设置了左右padding-left和padding-right后，将左右两个div用相对布局position: relative并分别配合right和left属性，以便左右两栏div移动后不遮挡中间div

    (2) 双飞翼布局

    双飞翼布局，为了中间div内容不被遮挡，直接在中间div内部创建子div用于放置内容，在该子div里用margin-left和margin-right为左右两栏div留出位置。
